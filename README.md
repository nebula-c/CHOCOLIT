# CHOCOLIT
CAEN High-voltage Online Control Operation & Logging Interface Tool
For VME HV module, V4718, USB



# Installation
```
git clone https://github.com/nebula-c/CHOCOLIT.git
cd CHOCOLIT/
pip3 install .     
```

# Run
You can run the following command from anywhere:
```
chocolit
```
